Ultra Nightmare

Clearly inspired by Wolfenstein 3D, this game is a First Person Shooter in which the player can control the view horizontally but not vertically, shoot enemies, pick up items from the ground and open sliding doors. Not F Zero

This game is coded in Java, using the game library LibGDX. After playing F-Zero on my old Super Nintendo Entertainment System I wondered how much of the game I could reproduce with today’s technology. I’ve always enjoyed watching the Mode 7 graphics. I just wanted to make a copy of the game by myself using Java. Another fan of the game from the LibGDX community MGSX reached out to me and contributed with loading time optimizations by using the GPU to generate/paint the track instead of the CPU which improved loading times by thousands of percents....
